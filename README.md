# Calculator DevOps Project

## 📌 Project Overview

This project demonstrates a simple web application deployed using DevOps practices.

## 🚀 Technologies Used

* HTML, CSS, JavaScript
* Docker
* GitHub Actions (CI/CD)
* AWS EC2

## 🐳 Docker

Application is containerized using Docker and served via Nginx.

## ⚙️ CI/CD Pipeline

* Code pushed to GitHub
* GitHub Actions triggers deployment
* Application deployed to EC2 using SSH
* Docker container is built and run
